================================================================================
KEY DIFFERENCES BETWEEN QUESTION TYPES:
================================================================================

1. TASK DESCRIPTION varies:
   - Route Planning: Focus on tracing navigation paths and turn sequences
   - Object Distance: Focus on finding all candidates and comparing distances
   - Object Direction: Focus on mental perspective-taking from a specific viewpoint

2. ANSWER HINT varies:
   - Route Planning: "Choose the correct sequence of turns"
   - Object Distance: "Choose the object that is closest"
   - Object Direction (easy): "Choose: left or right"
   - Object Direction (medium): "Choose: left, right, or back"
   - Object Direction (hard): "Choose: front-left, front-right, back-left, or back-right"

3. Rest of prompt is IDENTICAL across all question types

4. EXPLORATION vs FINAL STEP:
   - Exploration: Detailed movement instructions + examples + warnings
   - Final step: Simple "answer now" instruction
