# Color Palette

Project color palette for plots and visualizations.

| Name        | Hex       | Preview                                | Usage                                    |
|-------------|-----------|----------------------------------------|------------------------------------------|
| Red         | `#BD0A0A` | ![#BD0A0A](https://placehold.co/20/BD0A0A/BD0A0A) | 8B model lines, wrong answers            |
| Green       | `#229E4A` | ![#229E4A](https://placehold.co/20/229E4A/229E4A) | 4B model lines, correct answers          |
| Blue        | `#443ACF` | ![#443ACF](https://placehold.co/20/443ACF/443ACF) | Sequential pipeline, primary accent      |
| Violet      | `#DEDAFF` | ![#DEDAFF](https://placehold.co/20/DEDAFF/DEDAFF) | Light fill, background, secondary accent |
| Light Blue  | `#C6DCFF` | ![#C6DCFF](https://placehold.co/20/C6DCFF/C6DCFF) | Light fill, background, tertiary accent  |

## Python Usage

```python
COLORS = {
    "red":        "#BD0A0A",
    "green":      "#229E4A",
    "blue":       "#443ACF",
    "violet":     "#DEDAFF",
    "light_blue": "#C6DCFF",
}
```

## CSS Usage

```css
:root {
    --color-red:        #BD0A0A;
    --color-green:      #229E4A;
    --color-blue:       #443ACF;
    --color-violet:     #DEDAFF;
    --color-light-blue: #C6DCFF;
}
```

## LaTeX Usage

```latex
\definecolor{myred}{HTML}{BD0A0A}
```
